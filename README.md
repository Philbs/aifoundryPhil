# AI Foundry Course Portal 🚀

Este repositório contém o código fonte para o portal de treinamento de AI Foundry da Microsoft.

## Recursos:
- 4 Módulos de aprendizado.
- 120 slides detalhados.
- Downloads de arquivos PPTX integrados.
- Design moderno com Glassmorphism.

## Como rodar localmente:
1. `npm install`
2. `npm run dev`

## Publicação automática:
O site é publicado automaticamente via GitHub Actions no endereço:
https://Philbs.github.io/aifoundryPhil/
